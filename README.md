# password-tester
Алгоритм аналізу криптостійкості паролів, який може бути використаний під час розробки інформаційних систем або для підвищення обізнаності користувачів щодо створення надійних паролів. Запропонований алгоритм може бути інтегрований у системи автентифікації для зменшення кількості слабких та скомпрометованих парольних комбінацій.
